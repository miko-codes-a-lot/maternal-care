Maternal Care - User Manual

Empowering healthcare through technology, the Maternal Care application is an essential tool for Super Admins, Admins, Barangay Health Workers (BHW), and pregnant users. This manual provides a comprehensive guide to navigating the application's features, ensuring effective management of user data, scheduled checkups, health monitoring, and communication. Start today to deliver superior care for maternal health.

1. Login Session

The login session allows users to access the app securely. Super Admin has extended permissions to manage data and users.

2. Super Admin Menu

Super Admin has access to all features, including the ability to select addresses, manage users, and view session data.

3. Address Selection

Super Admin can select specific addresses to view related user data and sessions. Printing sessions is also enabled for administrative ease.

4. Sample User Data

Pregnant users' data is displayed in a dashboard accessible by the Super Admin, with relevant information on health and session history.

5. Admin User Features

The admin can create accounts for Barangay Health Workers (BHW). They can also manage settings and view sample users' profiles for management.

6. Settings Page

The settings page enables Super Admin to configure the app and manage users effectively. This includes adding or removing users, modifying app settings, and controlling access rights.

7. BHW Login

BHWs can securely log in to their dashboard to access user management tools and scheduled tasks.

8. BHW Menu UI

The BHW menu provides easy navigation to manage users, view reminders, and access settings.

9. Reminders for Scheduled Checkups

The application sends reminders to BHWs about upcoming prenatal checkups to ensure timely care for pregnant users.

10. Pregnant Users Management

BHWs can create, preview, and manage pregnant users, including their addresses and pregnancy numbers. Successfully added users are shown on the dashboard.

11. Condition Status

The application highlights user conditions with color-coded indicators: green for normal and red for critical.

12. Prenatal Checkup

BHWs can schedule checkups, monitor statuses, and update health information. BMI is calculated automatically after checkup data is submitted.

13. Immunization and Health Monitoring

BHWs manage immunization records and check the overall health of pregnant users during their care cycle.

14. Messages

BHWs can send and receive messages with pregnant users, ensuring effective communication and coordination.

15. BHW Settings

The settings page allows BHWs to configure their profiles and preferences within the application.

16. Pregnant Menu UI

The Pregnant Menu UI offers quick access to reminders, health updates, and condition monitoring for better prenatal care.

17. Pregnant User Reminder Alert

Sends timely alerts for checkups and immunizations to help maintain a healthy pregnancy.

18. Pregnant Reminder List

Displays upcoming reminders, ensuring users stay updated on their prenatal care schedules.

19. Pregnant Health Record

View and update health records, including checkup details and immunization history.

20. Pregnant Condition Status

Displays health status using indicators to identify critical or normal conditions.

21. Pregnant Trimester Checkup

Monitor and track trimester checkup schedules and updates for better pregnancy care.

22. Pregnant BMI Computation

Automatically calculates BMI to assess and track user health during pregnancy.

23. Pregnant Checkup Data

Stores and updates user checkup data, ensuring accurate health tracking.

24. Pregnant Settings

Allows pregnant users to customize app preferences and health monitoring settings.
